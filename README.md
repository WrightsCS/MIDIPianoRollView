MIDIPianoRollView
====

Customisable UIScrollView subclass for rendering/editing MIDI notes on a piano roll view.

----

![alt tag]()

Requirements
----

* iOS 9.0+
* Swift 4.2+
* Xcode 10.0+


Usage
----

Create a `MIDIPianoRollView` either from storyboard (subclass from `UIScrollView`) or programmatically.
